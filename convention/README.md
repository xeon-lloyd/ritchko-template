# Convention Guide

## 목적
- 이 폴더는 이 레포에서 공통으로 따를 코딩 컨벤션의 기본값을 모아둔 곳이다.
- 영역별 `AGENTS.md`, 체크리스트, 전용 컨벤션 문서에 없는 일반 규칙은 이 폴더 문서를 기준으로 판단한다.

## 문서 구성
- `naming.md`: 식별자 이름 규칙과 검색 가능한 네이밍 기준
- `comment.md`: 주석을 언제, 어떤 톤과 밀도로 쓰는지에 대한 기준
- `function.md`: 함수 책임 분리, 시그니처 유지, 부수효과 처리 기준
- `formatting.md`: 파일 배치, 함수 배치, 공백과 읽기 흐름 기준

## 읽는 순서
1. 루트 `AGENTS.md`
2. 작업 대상 영역의 `AGENTS.md`, 체크리스트, 전용 컨벤션 문서
3. 이 폴더의 관련 문서

## 우선순위
- 작업 대상 영역 문서에 더 구체적이거나 더 엄격한 규칙이 있으면 그 문서를 우선한다.
- 충돌이 없으면 `convention/` 문서를 기본값으로 적용한다.
- 별도 지시가 없으면 일반적인 외부 프레임워크 습관보다 이 폴더 문서를 우선한다.

## 적용 메모
- backend는 `backend/CONVENTION.md`에서 operation 구조, response 이름, enum, 가드 절, import 배치 같은 영역 전용 규칙을 추가로 정의한다.
- 특히 backend는 `comment.md`의 "주석은 적게" 원칙을 유지하되, 흐름이 바로 보이도록 짧은 한국어 단계 주석을 유지하는 예외가 있다.
